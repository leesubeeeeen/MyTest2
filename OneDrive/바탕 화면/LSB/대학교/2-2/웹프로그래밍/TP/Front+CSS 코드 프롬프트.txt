#만들어야 할 것 : 웹페이지 코드 중 html, css 코드 부분

내가 만들어야 할 페이지 사진을 그 페이지에 대한 설명과 함께 하나씩 보내줄텐데, html과 css 코드에 점진적으로 반영해서 나에게 코드를 따로 분리해서 줘. (html, css 파일 분리)

처음 보낸 사진은 시스템 구조도야. 내 웹 페이지가 어떤 기능을 어떤 구조로 제공하는지가 쓰여져 있지. 그리고 두번째 사진은 사이트맵이야. 세번째 사진은 사이트맵 상세설명이 적혀져 있어. 그런데 난 이 기능을 지금 다 구현할 생각은 아니고 front 부분만 만들려고 해. 그래서 이 시스템 구조도와 사이트맵, 사이트맵 상세설명을 활용해서 내가 어떤 웹을 만들고 싶은지를 잘 파악해주고 내가 앞으로 보내줄 웹페이지 사진을 기반으로 html, css코드를 작성해줘. 내가 "여기까지야" 라는 말을 하기 전까진 계속 수행해야하고 나에게 사진 달라고 요구해줘.
